# **App Name**: PokerPay

## Core Features:

- Ledger Import: Scrape the website for a CSV file that's stored under a download ledger button.
- Player Identification: Automatically identify all players involved in the ledger.
- Payment Calculation: Calculate the net amount owed/due to each player.
- Payment Summary: Display a clear summary of all payments to be made.
- Statement Generator: Use a generative AI tool to reason about payment information and create a human readable, customized payment statement for distribution to each player. This feature uses a tool.

## Style Guidelines:

- Primary color: Casino green (#00A36C) for a classic and sophisticated feel.
- Background color: Black (#000000) for a sleek and modern look.
- Accent color: Red (#FF0000) to highlight key information and actions.
- Body font: 'Inter' (sans-serif) for readability and a modern aesthetic.
- Headline font: 'Space Grotesk' (sans-serif) for a bold and modern look. To be paired with Inter.
- Use clear, simple icons to represent players, payments, and other actions.
- A clean, table-based layout for the ledger and payment calculations.
- Subtle transitions to indicate changes in payment status.
